# Fable Cup

The prediction market for World Cup 2026. Editorial, premium, zero gambling aesthetics.

Read `DEVELOPER-BRIEF.md` first. It covers setup, Supabase schema, deployment, the manual operations runbook, and the design rules this codebase must follow.

Quick start:

```
npm install
cp .env.example .env.local
npm run dev
```

The site runs in demo mode without Supabase credentials.
